📏 المسافات
padding من جوّا ← → margin من برّا
📦 padding — جوّاني

المسافة بين المحتوى وحافة العنصر

padding: 20px; /* كل الجهات */ padding: 10px 20px; /* فوق/تحت | يمين/شمال */ padding-top: 10px; /* جهة واحدة */
↔ margin — برّاني

المسافة بين العنصر وأي حاجة حواليه

margin: 20px; /* كل الجهات */ margin: 0 auto; /* توسيط أفقي ⭐ */ margin-bottom: 16px; /* جهة واحدة */

← padding (جوّه الإطار)
margin

🏷️ تاجات HTML
كل عنصر على الصفحة جاي من تاجة — دي الأساسية
div
صندوق عام — الأكتر استخداماً
مفيش معنى تلقائي — بتديه شكل بالـ CSS
<div class="card">
h1-h6
عناوين — h1 الأكبر والأهم
h1 واحدة بس في الصفحة — محركات البحث بتقرأها
<h2>عنوان فرعي</h2>
p
فقرة نص عادية
للنص الطويل — بياخد سطر كامل
<p>نص هنا</p>
a
لينك — href هو الرابط
target="_blank" يفتح في تاب جديد
<a href="https://...">
img
صورة — src=المسار / alt=وصف
alt مهم للـ SEO ولو الصورة مش ظهرت
<img src="photo.jpg" alt="وصف">
input
خانة إدخال — type بيحدد النوع
text / email / password / number
<input type="email">

Semantic HTML: في تاجات زي <nav> و<section> و<article> — معناها واضح للمتصفح ومحركات البحث. استخدمها بدل <div> لما ينفع.

🎯 Selectors وSpecificity
إزاي CSS بتختار العنصر — وليه أحياناً الكود مش بيتطبق!
النوعCSSHTMLقوةمعناه
التاجة h1 { color: blue; } <h1>نص</h1> 1 كل الـ h1
.class الأشيع .card { background: #fff; } class="card" 10 نقطة + الاسم
#id #header { height: 60px; } id="header" 100 هاش # — واحد بس
:hover .btn:hover { opacity: 0.8; } — 10 لما الماوس فوقه
جوّا بعض .card p { color: gray; } — 11 كل p جوّا .card
❓ ليه الكود مش بيتطبق؟
p { color: red; } /* قوة = 1 */ #main p { color: blue; } /* قوة = 101 — بيكسب! */
✅ القاعدة
الـ ID أقوى من الـ class، والـ class أقوى من التاجة.
لو كتبت خاصية وهي مش بتتطبق — على الأغلب في selector تاني أقوى منه بيكتب فوقه.

🧠 جرّب فاهم قد إيه
سؤالين بس — بدون ضغط
١. إيه الخاصية اللي بتغير لون النص نفسه؟
🔲 background-color
🔲 color
🔲 border-color
✅ صح! color بتغير لون النص — background-color بتغير الخلفية.
❌ مش كده — color هي اللي بتغير لون النص. background-color هي للخلفية.
٢. عايز توسّط صندوق أفقياً — إيه الـ CSS؟
🔲 padding: 0 auto
🔲 margin: 0 auto
🔲 text-align: center
✅ صح! margin: 0 auto مع width محددة = توسيط الصندوق أفقياً.
❌ مش كده — margin: 0 auto هو الصح. text-align للنص بس مش للصناديق.
