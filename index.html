الدرس الأول
🎨 الألوان
في CSS فيه 3 خصائص بس للألوان — اعرفهم واعرف تلاقيهم في أي كود
الخاصية مثال بتتحكم في إيه نوع
color color: #e53935; لون النص نفسه نص
background-color background-color: #fff8e1; لون خلفية العنصر كله خلفية
border-color border-color: #cccccc; لون الإطار حوالين العنصر إطار
صيغ الألوان #ff0000 = أحمر
rgb(255,0,0) = أحمر
red = أحمر 		3 طرق تكتب نفس اللون — الـ HEX (#) الأكتر استخداماً
استخدم Google "color picker" عشان تلاقي الكود 		مرجع

لو عايز تغير لون في أي كود: ابحث بـ Ctrl+F عن # أو عن كلمة color وهتلاقي كل الألوان في الصفحة على طول

الدرس التالت
📏 المسافات
padding من جوّا — margin من برّا
📦 padding — جوّاني

المسافة بين المحتوى وحافة العنصر من الداخل

padding: 20px; padding-top: 10px; padding: 10px 20px; /* فوق/تحت | يمين/شمال */

كبّر الـ padding → العنصر يكبر بس المحتوى مش بيتحرك

↔ margin — برّاني

المسافة بين العنصر وأي حاجة حواليه

margin: 20px; margin-bottom: 16px; margin: 0 auto; /* توسيط أفقي */

margin: 0 auto = أشهر سطر عشان توسّط أي صندوق

الخاصيةمثالاستخدام
gap gap: 16px; المسافة بين عناصر الـ flex/grid
line-height line-height: 1.6; /* 1.0 ضيق | 2.0 واسع */ المسافة بين سطور النص
الدرس الرابع
✍️ النص والخط
كل حاجة بتتحكم في شكل الكلام في الصفحة
01
font-family — نوع الخط
بيحدد شكل الحروف نفسها
font-family: Arial, sans-serif;
02
font-weight — سُمك الخط
normal / bold / أرقام من 100 لـ 900
font-weight: 700;
03
text-align — محاذاة
right / center / left
text-align: center;
04
text-decoration — خط تحت
شيل السطر من اللينك أو حطّه
text-decoration: none;
05
letter-spacing — مسافة الحروف
للعناوين المكتوبة بخط رفيع
letter-spacing: 0.1em;
الدرس الخامس
🔲 الإطار والزوايا
الخط اللي حوالين العنصر والزوايا الدايرية
الخاصيةمثالبتعمل إيه
border border: 1px solid #ccc; /* السُمك | النوع | اللون */ border: none; /* شيل الإطار */ إطار حوالين العنصر
solid=خط / dashed=متقطع / dotted=نقط
border-radius border-radius: 4px; /* شوية */ border-radius: 12px; /* أكتر */ border-radius: 50%; /* دايرة */ تدوير الزوايا — 0=حادة / 50%=دايرة كاملة
border-bottom border-bottom: 2px solid #333; إطار من جهة واحدة بس
border-top / left / right / bottom
box-shadow box-shadow: 0 2px 8px rgba(0,0,0,0.15); /* x y blur لون+شفافية */ ظل تحت أو حوالين العنصر

نصيحة: لما بتشوف عنصر مستدير زوايا في أي موقع — دور على border-radius في CSS وغيّر الرقم

الدرس السادس
📦 الترتيب
إزاي العناصر بتتوزع جنب بعض — Flexbox هو الأكتر استخداماً
Flexbox — جنب بعض
display: flex; flex-direction: row; /* جنب بعض (افتراضي) */ flex-direction: column; /* فوق بعض */ justify-content: center; align-items: center;
Grid — شبكة
display: grid; grid-template-columns: 1fr 1fr; /* عمودين */ 1fr 1fr 1fr; /* 3 أعمدة */ gap: 16px; /* مسافة بين الخلايا */
الخاصيةالقيمبتعمل إيه
justify-content center / space-between / flex-end توزيع عناصر الـ flex أفقياً
align-items center / flex-start / stretch محاذاة عناصر الـ flex رأسياً
position relative / absolute / fixed طريقة تموضع العنصر — absolute فوق كل حاجة

الدرس التامن
🎯 CSS Selectors
إزاي CSS بتقول "العنصر ده بالظبط اللي عايز أغيره"
النوع في CSS في HTML معناه
التاجة h1 { color: blue; } <h1>عنوان</h1> كل الـ h1 في الصفحة
.class الأكتر .card { background: #fff; } <div class="card"> نقطة . + الاسم في CSS
#id #header { height: 60px; } <div id="header"> هاش # — عنصر واحد بس
:hover .btn:hover { background: #333; } — لما الماوس يبقى فوق العنصر
جوّا بعض .card p { color: gray; } — كل <p> جوّا .card

القاعدة: اسم الـ class في HTML بيطابق .اسم في CSS تماماً — لو غيّرت الاسم في واحدة اتغيّر في التانية

الدرس التاسع
👁️ الإخفاء والظهور
3 طرق مختلفة تخلي حاجة تختفي — لكل واحدة لها استخدام
الخاصيةمثالالفرق
display: none display: none; مخفي تماماً — مش بياخد مساحة — JavaScript بتستخدمه الأكتر
display: block display: block; يظهر تاني — عكس none
opacity: 0 opacity: 0; /* شفاف */ opacity: 0.5; /* نص شفافية */ opacity: 1; /* واضح */ شفاف بس لا يزال بياخد مساحته
visibility: hidden visibility: hidden; مخفي بس المساحة فاضية — نادر الاستخدام
overflow overflow: hidden; /* اقطع الزيادة */ overflow: scroll; /* اعمل scroll */ إيه اللي يحصل لو المحتوى أكبر من العنصر
transition transition: all 0.3s ease; انيميشن ناعم لما أي خاصية تتغير

<!DOCTYPE html>
<html>
<head>
<style>

  body {
    font-family: Arial, sans-serif;
    background: #f5f5f5;
    padding: 40px;
  }

  h1 {
    color: #1a73e8;     /* ← غيّر اللون هنا */
    font-size: 32px;
    text-align: center;
  }

  p {
    font-size: 15px;    /* ← كبّر الحجم هنا */
    color: #444;
    line-height: 1.7;
  }

  .card {
    background: white;
    border: 1px solid #ddd;
    border-radius: 8px;   /* ← دوّر الزوايا */
    padding: 24px;
    max-width: 500px;
    margin: 0 auto;
  }

  .btn {
    background-color: #1a73e8; /* ← غيّر اللون */
    color: white;
    padding: 10px 24px;
    border: none;
    border-radius: 6px;
    cursor: pointer;
  }

</style>
</head>
<body>

  <div class="card">
    <h1>مرحباً بالعالم</h1>
    <p>ده نص تجريبي. جرّب تغيّر الحجم واللون.</p>
    <button class="btn">اضغط هنا</button>
  </div>

</body>
